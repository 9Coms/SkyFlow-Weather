# 🌌 SkyFlow - Weather & Map Dashboard

ระบบแดชบอร์ดพยากรณ์อากาศและแผนที่อัจฉริยะสุดพรีเมียม ออกแบบมาด้วยธีมมืด (Dark Mode) สุดล้ำพร้อมลูกเล่น Glassmorphism หรูหรา สะดวก รวดเร็ว และรองรับทุกขนาดหน้าจอ

## 🚀 ฟีเจอร์หลัก (Features)
*   📍 **Interactive Map (Leaflet.js)**: จิ้มและลากหมุดบนแผนที่เพื่อดูข้อมูลสภาพอากาศได้ทันทีทั่วโลก
*   🔍 **Smart Search & Autocomplete**: ค้นหาตามชื่อเมือง จังหวัด หรือประเทศด้วย API ของ Nominatim ค้นหาพิกัดแบบเรียลไทม์
*   🛰️ **Geolocation Integration**: ดึงพิกัดปัจจุบันของคุณผ่าน GPS เพื่อรายงานสภาพอากาศได้ทันทีตั้งแต่เปิดหน้าเว็บ
*   📊 **Visual Trends (Chart.js)**: กราฟแสดงการเปรียบเทียบแนวโน้มอุณหภูมิร่วมกับโอกาสเกิดฝนในอีก 12 ชั่วโมงล่วงหน้า
*   🕐 **Hourly Forecast**: รายงานสภาพอากาศรายชั่วโมงล่วงหน้า 24 ชั่วโมงในรูปแบบ Carousel สไลด์ปัดใช้งานง่าย
*   📅 **7-Day Forecast**: พยากรณ์ล่วงหน้า 7 วัน พร้อมระบุค่าอุณหภูมิสูงสุด-ต่ำสุดและสภาพอากาศโดยทั่วไป
*   ☀️ **UV & Solar Info**: แสดงข้อมูลดัชนี UV สูงสุดพร้อมระดับเตือนภัย และเวลาขึ้น-ตกของดวงอาทิตย์
*   💖 **Credits & Attribution**: โซนผู้พัฒนาที่มุมด้านล่างแบบเรืองแสงสุดเจ๋งสำหรับ **COMCOM**

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
*   **HTML5 & Vanilla CSS**: โครงสร้างและดีไซน์แบบ Custom ทั้งหมด ไม่พึ่งพา TailwindCSS เพื่อความลื่นไหลสูงสุด
*   **Vanilla JavaScript (ES6)**: จัดการ Logic หน้าบ้าน ดึงข้อมูล และจัดการ DOM ทั้งหมด
*   **Leaflet.js API**: ระบบแผนที่โต้ตอบ
*   **Chart.js**: กราฟแสดงข้อมูลพยากรณ์อัจฉริยะ
*   **Open-Meteo API**: แหล่งข้อมูลพยากรณ์อากาศแบบสาธารณะที่แม่นยำสูง
*   **Nominatim (OpenStreetMap)**: ระบบแปลงพิกัดและค้นหาข้อมูลที่อยู่ (Geocoding & Reverse Geocoding)
*   **Lucide Icons**: ไอคอนหลากหลายครอบคลุมแบบเบาสบายตา

## 📂 โครงสร้างโปรเจกต์
```
wasd/
├── index.html       # โครงสร้างหน้าเว็บหลักและ UI
├── style.css        # สไตล์ชีท ตกแต่งดีไซน์ Glassmorphic, Responsive และ Animation ต่างๆ
├── app.js           # โค้ด Logic เชื่อมต่อ API แผนที่ และจัดการการแสดงผลทั้งหมด
└── README.md        # เอกสารอธิบายการใช้งาน (ไฟล์นี้แหละ)
```

## 💻 การติดตั้งและเปิดใช้งาน (Setup & Usage)
โปรเจกต์นี้เขียนด้วย **Vanilla HTML/CSS/JS** เพียว ๆ ไม่มีขั้นตอนการติดตั้งที่ซับซ้อนให้ยุ่งยากหัวจะปวด:

1.  **ดาวน์โหลด/คัดลอกไฟล์ทั้งหมด** ไปไว้ในเครื่องของคุณ
2.  **เปิดใช้งาน**:
    *   **วิธีที่ 1 (ง่ายสุด)**: ดับเบิ้ลคลิกไฟล์ `index.html` เพื่อรันในเว็บเบราว์เซอร์ได้ทันที
    *   **วิธีที่ 2 (แนะนำสำหรับ Local Server)**: เปิดรันผ่านเซิร์ฟเวอร์จำลองเพื่อหลีกเลี่ยงข้อจำกัดบางประการของเว็บ
        *   ด้วย Python (3.x):
            ```bash
            python -m http.server 8000
            ```
            แล้วเปิดเข้าลิงก์ `http://localhost:8000`
        *   ด้วย Live Server (VS Code Extension): คลิกขวาที่ `index.html` แล้วเลือก "Open with Live Server"
